# Core Skills — Guia

## Objetivo

Este diretório é reservado para playbooks operacionais reutilizáveis.

## Exemplos de skills

- `generate-demand/SKILL.md` (padrão de mercado)
- `generate-demand.md` (compatibilidade)
- `implement-plugin.md`
- `bugfix-api.md`
- `review-pr.md`
- `refactor-safe-minimal.md`

## Regras de uso

- Skills não substituem regras globais.
- Skills devem ser aplicadas dentro de `core/workflow.md`.
- Em conflito, prevalecem `core/rules.md` e as fontes canônicas do projeto.
